# BookVault Client

This is the frontend application for the BookVault project. It connects to the REST API and allows users to manage their book collection.

---

## 🚀 How to Run

### 1. Install dependencies

```bash
npm install
```

### 2. Start the client

```bash
npm start
```

The client will run at:

```
http://localhost:5500
```

---

## ⚠️ Important

Make sure the backend API is running at:

```
http://localhost:3000
```

---

## 📚 Features

* View all books from the API
* Search books by title, author, or genre
* Filter books by reading status
* Add new books
* Edit book details
* Delete books
* View detailed book information

---
